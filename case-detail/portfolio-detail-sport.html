Планы проекта

Разработка единого спортивного портала России для популяризации спортивного образа жизни. Цель портала - минимальная точка входа любого человека в спорт. Портал позволит подобрать спорт для себя, найти ближайший центр для занятий и вести учет посещенных спортивных мероприятий.
